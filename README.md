# AgroSort Machine
AgroSort Machine - project for [RoboArt 2026](https://roboart36.ru) and [Detalka 2026](https://detalka-fest.ru) robotics contests by [Robo_Start](https://robo-start.ru/#contact) and [JohnnyTheBeerDevourer](https://johnnythebeer.dev/#contact) (Members of [RoboTrack club](https://робототехника-новороссийск.рф))

### Repository Map
- This repository is PyCharm project with special subdirectories
- [Docs](https://github.com/RoboStartCoder/agro-sort-machine/tree/main/docs) - Documents and Presentation of project
- [Arduino](https://github.com/RoboStartCoder/agro-sort-machine/tree/main/arduino) - Machine code (PlatformIO)
- [Web](https://github.com/RoboStartCoder/agro-sort-machine/tree/main/web) - Web UI for control machine

### Shortly. How it works
Machine is a conveyor with 3 servos and camera at start. Camera detect type, color and size of object. Then servos rotate for move object to suitable container
